# openclaw-upgrade

> Alias package for **[openclaw-cli](https://www.npmjs.com/package/openclaw-cli)**.

This package runs the same CLI engine as `openclaw-cli`.

## Install

```bash
npm install -g openclaw-upgrade
```

## Usage

```bash
openclaw-upgrade watch -d
openclaw-upgrade status
openclaw-upgrade gateway restart
```
