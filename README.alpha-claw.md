# alpha-claw

> Alias package for **[openclaw-cli](https://www.npmjs.com/package/openclaw-cli)**.

This package runs the same CLI engine as `openclaw-cli`.

## Install

```bash
npm install -g alpha-claw
```

## Usage

```bash
alpha-claw watch -d
alpha-claw status
alpha-claw gateway restart
```
